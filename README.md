### To run the API
```bash
pip install -r requirements.txt
# to install requirements packages
```
```bash
fastapi dev main.py
```

### To create virtual python environment

```bash
python -m venv .venv
# only on the first setup
```
```bash
source .venv/bin/activate
# every time you program
```
```bash
which python
```
```bash
python -m pip install --upgrade pip
# only on the first setup
```
```bash
echo "*" > .venv/.gitignore
```
```bash
deactivate
# to get out of venv
```
